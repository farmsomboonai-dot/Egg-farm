# ระบบขายไข่ฟาร์มสมบูรณ์ (เอสเจเอฟ ฟาร์ม)

เว็บแอปต้นแบบ — **ไฟล์สถิตล้วน ไม่ต้องบิลด์ ไม่ต้องติดตั้ง Node**
React + lucide-react โหลดผ่าน CDN และแปลง JSX ในเบราว์เซอร์ด้วย Babel

---

## 📁 โครงสร้างไฟล์

```
app/
├─ index.html     ← "เปลือก" หน้าเว็บ: โหลดไลบรารี + เรียก egg-sales.jsx (เล็ก ไม่ค่อยต้องแก้)
├─ egg-sales.jsx  ← โค้ดแอปทั้งหมด ★ แก้ไขหน้าตา/ฟังก์ชันที่ไฟล์นี้ที่เดียว
├─ serve.ps1      ← เซิร์ฟเวอร์ทดสอบบนเครื่อง (PowerShell, ไม่ต้องลงอะไร)
└─ README.md      ← ไฟล์นี้
```

> แก้โค้ด = เปิด `egg-sales.jsx` แก้แล้วเซฟ → กด refresh ในเบราว์เซอร์ เห็นผลทันที (ไม่ต้องบิลด์)

---

## ▶️ รันบนเครื่อง (ทดสอบ)

เปิด PowerShell ในโฟลเดอร์ `app` แล้วสั่ง:

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
```

เปิดเบราว์เซอร์ที่ **http://localhost:5173** · หยุดด้วย `Ctrl+C`
ถ้าพอร์ตชนกับโปรแกรมอื่น ใช้พอร์ตอื่นได้: `powershell -ExecutionPolicy Bypass -File serve.ps1 5174`

> ⚠️ ต้อง “เปิดผ่านเซิร์ฟเวอร์ (http)” เท่านั้น — ดับเบิลคลิกเปิดไฟล์ตรงๆ (`file://`) จะไม่ทำงาน
> เพราะเบราว์เซอร์บล็อกการโหลดโมดูล และต้องต่ออินเทอร์เน็ตได้ (โหลดไลบรารีจาก CDN)

---

## 🚀 เอาขึ้นเซิร์ฟเวอร์ / โฮสติ้ง

แอปนี้เป็น **ไฟล์สถิต** จึงเอาขึ้นที่ไหนก็ได้ แค่อัปโหลด 2 ไฟล์: `index.html` และ `egg-sales.jsx`
(ไฟล์ `serve.ps1` กับ `README.md` ไม่ต้องอัปก็ได้)

### ตัวเลือก 1 — โฮสต์สถิตฟรี (ง่ายสุด)
- **Netlify / Vercel / Cloudflare Pages:** ลากโฟลเดอร์ `app` ไปวางในหน้าเว็บ deploy ได้เลย
- **GitHub Pages:** push ไฟล์ขึ้น repo แล้วเปิด Settings → Pages

### ตัวเลือก 2 — เว็บเซิร์ฟเวอร์ของตัวเอง (Apache / Nginx / IIS / โฮสติ้งทั่วไป)
อัปโหลด `index.html` + `egg-sales.jsx` ไปไว้ใน document root (เช่น `public_html/`) เปิดผ่านโดเมนได้เลย
ไม่ต้องตั้งค่าพิเศษ — เป็น HTML/JS ธรรมดา

### ข้อควรรู้ของเวอร์ชันนี้
- พึ่งอินเทอร์เน็ต (โหลด React/ไอคอน/Babel จาก CDN) และแปลง JSX ตอนเปิดหน้า → โหลดครั้งแรกช้ากว่าปกติเล็กน้อย
- เหมาะกับ **ทดลอง/เดโม/ใช้ภายใน** ข้อมูลยังเป็นค่าสมมติในหน่วยความจำเบราว์เซอร์ (refresh แล้วรีเซ็ต — ยังไม่มีฐานข้อมูล)

## ⏫ อัปเกรดเป็นเวอร์ชัน Production (เมื่อพร้อม)

ถ้าจะใช้งานจริงจังบนเซิร์ฟเวอร์ แนะนำย้ายไป **Vite** เพื่อ:
รวมไลบรารีมาในตัว (ไม่พึ่ง CDN) · บิลด์ล่วงหน้า (โหลดเร็ว) · เปิดออฟไลน์ได้

ขั้นตอนคร่าวๆ (ต้องติดตั้ง [Node.js](https://nodejs.org) ก่อน):

```bash
npm create vite@latest egg-app -- --template react
# เอา egg-sales.jsx ไปไว้ใน src/ แล้วให้ src/main.jsx import มาแสดง
npm install lucide-react
npm run build      # ได้โฟลเดอร์ dist/ → อัปโหลดขึ้นเซิร์ฟเวอร์
```

> อยากให้ช่วยตั้งโครงสร้าง Vite ให้พร้อม (ไว้รัน `npm run build` ทันทีที่ลง Node) บอกได้เลย
