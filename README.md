# 🥚 ระบบขายไข่ — ฟาร์มไข่สมบูรณ์ (SJF Farm)

ต้นแบบระบบ POS/จัดการฟาร์มไข่ สำหรับ **บริษัท เอสเจเอฟ ฟาร์ม จำกัด** — ขายไข่ · ออกบิล · คลังไข่รายวัน · ผลผลิตรายหลัง · ระบบแผงไข่ · บัญชี/ลูกหนี้

เขียนด้วย **React** (ไฟล์เดียว, ไม่ต้อง build) รันผ่าน CDN + Babel standalone — เปิดได้ทันทีโดยไม่ต้องติดตั้ง Node

## โครงสร้าง
```
app/
  index.html      เชลล์ (importmap: react/lucide จาก esm.sh + @babel/standalone)
  egg-sales.jsx   ★ ซอร์สหลักไฟล์เดียว (แก้ที่นี่ที่เดียว รีเฟรชเบราว์เซอร์)
  serve.ps1       static server (.NET TcpListener) → http://localhost:5173
  README.md       วิธีรัน/deploy
extracted/        ต้นฉบับ + system-spec.docx (สเปกความต้องการ)
```

## วิธีรัน
```powershell
powershell -ExecutionPolicy Bypass -File "app/serve.ps1"
# เปิด http://localhost:5173
```

## ฟีเจอร์หลัก
- **ขายไข่** — เลือกลูกค้า · ตะกร้า · มัดจำแผง (ดำ/ส้ม ×1.1) · ส่วนลด/ค่าขนส่ง/หัก ณ ที่จ่าย · ออกบิล + ตัดสต็อก (responsive web/มือถือ)
- **ผลผลิตรายวัน** — กรอกไข่ดี(เบอร์)/ตกเกรดรายหลัง · สุ่มตรวจตอกไข่ · ย้อนดูรายวัน · เข้าสต็อกอัตโนมัติ
- **คลังไข่รายวัน** — rolling: คงเหลือ = ยกมา + ผลิตวันนี้ − ขายวันนี้
- **ประวัติบิล** — กรองช่วงวันที่/สถานะ · สรุปยอด · Export Excel ให้แผนกบัญชี
- **แผงไข่** — ยืม/รับคืน/คัดแยก/ทดแทน
- **บัญชี/ลูกหนี้ · แดชบอร์ด**

ข้อมูลปัจจุบันเก็บใน `localStorage` (ต้นแบบ) — กำลังย้ายขึ้น **Supabase** เพื่อเก็บถาวร/หลายผู้ใช้
